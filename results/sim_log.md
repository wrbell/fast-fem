# Simulation Log

Chronological journal of every simulation run. Each entry links to its full validation report in `results/`.

---

## How to Use

After each simulation, add an entry below. Copy this template:

```markdown
### [SIM-XXX] Title — Week X, Day X
**Date:**
**Analysis type:**
**Model:** (brief description)
**Mesh:** (element type, count)
**Key result:** (e.g., max stress = 142 MPa)
**Hand calc:** (e.g., σ = F/A = 140 MPa)
**Error:** X%
**Validation report:** [link](results/week_XX/sim_XXX_report.md)
**Lesson learned:**
```
